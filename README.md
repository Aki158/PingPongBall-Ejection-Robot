## 📝説明

このロボットは、ピンポン玉射出ロボットです。

とあるロボットコンテストで使用しました。

コンテストでは、正確性とスピードを評価されます。

コンテストのルールは、すべての的(1~9)にピンポン玉を入れるとクリアになります。

(詳細なルールの記載は、省略します)

基本的な機能として、ライントレース/ピンポン玉の刈り取り/射出ができます。

ロボットの動きとしては、ライントレースでロボットを移動させ所定の位置をセンサが検知すると、一旦停止してピンポン玉を刈り取ります。

その後、ライントレースを再開して的の前まで到着すると一旦停止しピンポン玉を射出します。

これを全ての的にロボットが、ピンポン玉を入れるまで繰り返されます。

## 👀プロジェクトについて

このプロジェクトは2人で開発しました。

ロボットのデザインから筐体の制作、3つの機能(ライントレース、刈り取り、射出)に関わる設計/実装/テストは、2人で1から作り上げました。
